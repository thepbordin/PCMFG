  <behavior>
    - plot_cluster(trajectories, cluster_result, cluster_id, output_path):
      Filters trajectories to only those in the specified cluster (using cluster_result.assignments)
      Extracts barycenter for this cluster from cluster_result.barycenters[cluster_id]
      Creates 9x2 grid (same layout as plot_overlay) but only showing cluster members
      Members colored with their cluster color from CLUSTER_COLORS, alpha=0.3
      Barycenter drawn as solid line (linewidth=3.5), diamond markers, alpha=1.0, zorder=10
      Title includes cluster label and member count
      Saves to output_path
    - plot_all(trajectories, output_dir, cluster_result=None):
      Creates output_dir if not exists
      Generates full suite of PNGs with overlay_ prefix:
        - overlay_grid.png (via plot_overlay)
        - overlay_{emotion}.png for each of 9 emotions (via plot_emotion)
        - overlay_atob.png, overlay_btoa.png (via plot_direction)
        - overlay_cluster_{id}.png for each cluster (via plot_cluster) — only if cluster_result provided
      Returns list[Path] of all generated file paths
      Uses Path(output_dir) for path construction
    - plot_cluster raises ValueError if cluster_id >= cluster_result.n_clusters
    - plot_cluster raises ValueError if cluster has 0 members
  </behavior>
